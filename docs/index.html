GooD Autobackuper

Automated Backup System for Google Drive

Relevance

Information technologies are based on data storage. The main method of ensuring security is data backup - in case of failures, attacks, for data recovery. Backup systems work with different protocols, platforms, file systems, different levels of complexity of backup rules, schedules, etc.

But sometimes such systems are unnecessary - this program makes a backup copy every day at a specified time in Google Drive.

Get started

Purpose

Purpose: to develop a program for automatic data backup to Google Drive cloud storage using the Python programming language.

Methodology:

The following was chosen to implement the task
Google

Popular cloud storage. Has an open API for user programs: Google Drive API; Google Cloud Console.

Python

The API supports several languages, including Python.

PyQt5

The Qt - PyQt5 framework was chosen for the graphical interface.

Pyinstaller

The Pyinstaller module is selected to create executable programs.

Pickle

Pickle is selected for Google authorization.

  1. Analyzed modern backup systems.
  2. Functional and non-functional requirements for the program are defined, in general: the ability to create (and delete) a schedule, rules for the schedule; the ability to log in to Google and copy data to Google Drive.
  3. The automatic backup program was designed in accordance with the above methodology and program requirements.
  4. Software-implemented automated backup system for Google Drive.
